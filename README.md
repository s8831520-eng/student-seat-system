# 可愛一班學生抽座位系統

這是一個為教室座位管理設計的網頁應用程式，專為「可愛一班」設計，但可自訂為任何班級使用。

## 功能特色

- ✅ 自訂班級名稱
- ✅ 保護學生個資（姓名中間字用O替代）
- ✅ 支援手機、平板、電腦等各種設備
- ✅ 響應式設計，自動適應不同螢幕大小
- ✅ 可設定不坐人座位
- ✅ 可手動指定固定座位
- ✅ 隨機分配座位（固定座位保持不變）
- ✅ 男生女生座位用不同顏色區分

## 使用說明

1. **設定班級名稱**：在標題區輸入框修改班級名稱
2. **產生座位表**：設定行數和列數，點擊「產生座位表」
3. **載入學生資料**：輸入學生資料（格式：座號 姓名 性別）
4. **管理座位**：
   - 標記不坐人座位
   - 手動指定固定座位
   - 隨機分配座位
5. **操作座位**：點擊座位可分配或移除學生

## 學生資料格式

每行一筆，用空白分隔：# student-seat-system
